1. What is a computer directory structure?  

A. The directory structure is a file organizational system that has a hierarchical structure, similar to a tree and its roots or an umbrella.   

Each directory and folder have its own unique path that establishes where they belong on a storage device.  

2. What is the root directory?  

A. The root directory is the head or top-level file in a computer directory system. The root directory contains all your other files and subdirectories.  

3. What is the purpose of subdirectories?  

A. The purpose of a sub directory is to store and manage additional subdirectories and files in accordance with its root directory.  

4. What is a file path?  

A. The file path is the name of a file or folder that identifies the location of information within a directory structure system 

5. Why is the directory structure essential for organizing and managing files on a computer?  

A. Directory structures are vital for tracking and streamlining your file paths. A simple and clean file system allows both software and the user to locate important files and information  

6. What is HTML?  

A. HTML stands for Hypertext Markup Language, and it serves as the foundation for all web pages/sites  

7. What is the purpose of HTML?  

A. HTML's purpose is to be a provide a standard language for web browsers and holds all the design elements together  

8. What is the difference between HTML and CSS?  

A. HTML serves as the structure for a webpage, while Cascading Style Sheets (CSS) utilize that foundation to enhance the visuals and styles for your webpage

12. Explain the steps you would take to create an external link to
another page.  

  1.  Open Dreamweaver and open your webpage       
  2.  Open Insert tab                   
  3.  Click  on "Hyperlink" 
  4.  Insert your type in the text box   
  5.  Insert Hyperlink into the link box  

Alternatively

  1.  Open Dreamweaver and open your webpage       
  2.  Type <a href="link">text</a> into your code.                       
  3.  Change "link" with your Hyperlink    
  4.  Change "text" with the type you want to see on the webpage ex: "Click for more information"
